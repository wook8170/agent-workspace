# 🍅 초등학생용 뽀모도로 타이머

`TEST3-7` 이슈 대응 — 초등학생을 위한 뽀모도로 타이머 웹앱입니다.

## 핵심 흐름

| 단계 | 시간 | 전환 조건 |
|------|------|----------|
| 집중 🐰 | 25분 (조정 가능 5–30분) | 타이머 종료 → 짧은 휴식 |
| 짧은 휴식 🐶 | 5분 (조정 가능 1–10분) | 타이머 종료 → 다음 집중 (대기) |
| 긴 휴식 🦄 | 15분 고정 | 집중 4회 완료 후 자동 전환 |

- 집중 4회가 한 **라운드**. 라운드 완료 후 긴 휴식(15분)으로 자동 전환.
- 전환마다 Web Audio API 효과음과 친근한 캐릭터 알림 팝업 표시.
- 오늘 완료한 세션 수를 🍅 토마토 이모지로 시각화.
- 진행 상황은 `localStorage`에 저장 (자정 자동 초기화).

## 실행

정적 파일만 사용하므로 별도 빌드 없이 실행 가능합니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173` 을 열면 타이머 화면이 표시됩니다.
