# 🍅 초등학생용 뽀모도로 타이머

초등학생을 위한 뽀모도로 타이머 웹앱입니다. 공부 집중 시간과 휴식 시간을 자동으로 관리하며, 친근한 캐릭터 알림과 효과음으로 아이들의 학습 루틴을 도와줍니다.

## 주요 기능

- **뽀모도로 사이클**: 집중(기본 25분) → 짧은 휴식(기본 5분) × 4회 → 긴 휴식(15분) 자동 전환
- **시각 타이머**: 원형 진행 링과 큰 글자 남은 시간 표시
- **효과음**: 집중 종료·휴식 종료·라운드 완료 시 Web Audio API 효과음
- **캐릭터 알림**: 공부(🐰) · 짧은 휴식(🐶) · 긴 휴식(🦄) 전환 알림
- **세션 기록**: 오늘 완료한 세션 수를 🍅 아이콘으로 시각화, localStorage에 날짜별 저장
- **집중·휴식 시간 조정**: 스텝퍼 버튼으로 집중 5–30분 / 휴식 1–10분 조정 가능
- **접근성**: WCAG 2.1 AA 색상 대비, 터치 영역 최적화, ARIA 레이블

## 기술 스택

- React 18 (UMD CDN) + Babel Standalone — 빌드 도구 없이 단일 HTML 파일로 실행
- Web Audio API — 효과음 자산 파일 없이 실시간 생성
- localStorage — 날짜별 세션 영속성

## 실행

정적 파일만 사용하므로 아래 명령으로 루트 디렉터리에서 바로 확인할 수 있습니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173` 을 열면 타이머 화면이 표시됩니다.
