# 초딩용 뽀모도로 타이머 — 아키텍처 제약사항

## NFR-001 / OPS-001: 서버 운영 항목 제외 (범위 외)

이 앱은 **외부 백엔드 없는 순수 클라이언트 SPA**입니다.
아래 항목은 이 프로젝트 범위에 포함되지 않습니다.

| 항목 | 이유 |
|------|------|
| 서버 측 오류 추적 (예: Sentry 서버 연동) | 외부 API 엔드포인트 없음 |
| 서버 알림 / 푸시 (예: FCM, APNs) | 백엔드 서버 없음 |
| 서버 이벤트 재처리 / 큐 | 서버리스 클라이언트 전용 앱 |
| 외부 분석 수집기 (서버 ping) | 외부 연동 없음 |
| 사용자 인증 / 세션 서버 | 로그인 기능 미포함 |

## 허용되는 클라이언트 전용 저장소

- `localStorage` — 세션 완료 횟수, 타이머 설정값 영구 보존
- `Web Audio API` — 알림음 재생 (브라우저 내장)
- CSS transition / 브라우저 애니메이션 — UI 상태 전환

## 기술 스택 요약

```
React + TypeScript (SPA)
브라우저 localStorage  ← 유일한 영속성 레이어
외부 HTTP 호출 없음
외부 npm 서버 런타임 패키지 없음
```
